## 📸 Image Upload Flow

```
1. User clicks 📷 button
   ↓
2. File picker opens
   ↓
3. User selects image
   ↓
4. Preview appears:
   ┌──────────────┐
   │ [Image]   ✕  │
   └──────────────┘
   ↓
5. User clicks send
   ↓
6. Message sent with image
   ↓
7. AI analyzes and responds:
   "I've analyzed your crop image.
    Disease: Leaf Blight
    Severity: Moderate
    Treatment: Apply fungicide..."
```

## 🎤 Voice Input Flow

```
1. User clicks 🎤 EN/HI button
   ↓
2. Microphone activates
   ↓
3. User speaks
   ↓
4. Speech converted to text
   ↓
5. Text appears in input field
   ↓
6. User can edit or send
```
